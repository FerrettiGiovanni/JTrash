## Avvio del Software

Per avviare JTrash, segui questi passaggi:

1. Assicurati di aver installato Java sul tuo sistema.

2. Scarica il repository JTrash dalla pagina principale del progetto su GitHub.

3. Esegui i seguenti comandi nel terminale o nel prompt dei comandi:

```bash
cd JTrash       # Assicurati di essere nella directory principale del progetto
cd src          # Vai alla directory "src"
cd controller   # Vai alla directory "controller" dove si trova la classe JTrash.java che contiene il metodo main
JTrash.java     # Avvia l'applicazione JTrash
